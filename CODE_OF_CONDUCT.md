# Code of Conduct

KnowledgeCivic is a public learning space. Participants must treat one another with dignity, assume good faith while checking claims carefully, and make room for people with different experiences and abilities.

Unacceptable behaviour includes harassment, discrimination, threats, doxxing, deliberate deception, plagiarism, manipulation of review processes, and retaliation against good-faith criticism.

Report concerns privately to the maintainers through the repository security or contact channel. Maintainers may edit, reject, or remove contributions that violate this code, and will document significant enforcement decisions without exposing personal data.
